获取网关信息

支持获取网关的型号、MAC地址以及固件版本。

示例

1import cassiasys
2
3model = cassiasys.gateway_type()
4mac = cassiasys.gateway_mac()
5ver = cassiasys.gateway_ver()
6
7print(model, mac, ver)

API

cassiasys.gateway_type() → str

网关型号，例如 "M2000"

cassiasys.gateway_mac() → str

网关MAC地址，例如 "CC:1B:E0:E4:C8:94"`

cassiasys.gateway_ver() → str

网关固件版本号，例如 "2.2.mp.2509031847"`
