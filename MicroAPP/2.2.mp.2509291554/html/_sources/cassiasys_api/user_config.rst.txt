API
=====

.. function:: cassiasys.save_user_config(config: str)
   
   保存用户配置，重启后也有效。保存失败抛出异常，比如超过16K大小。

   :parameters:
      - config: 配置内容

.. function:: cassiasys.read_user_config() -> str

   读取用户配置。读取失败抛出异常。

   :returns: 
      - config: 配置内容
