====================
获取网关信息
====================
支持获取网关的型号、MAC地址以及固件版本。

示例
=====

.. literalinclude:: ../../../../../../example/cassiasys/gateway_info.py
   :language: python
   :linenos:

API
=====

.. function:: cassiasys.gateway_type() -> str

   网关型号，例如 ``"M2000"``

.. function:: cassiasys.gateway_mac() -> str

   网关MAC地址，例如 ``"CC:1B:E0:E4:C8:94"```

.. function:: cassiasys.gateway_ver() -> str

   网关固件版本号，例如 ``"2.2.mp.2509031847"```
