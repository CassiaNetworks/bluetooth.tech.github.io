=====
获取网关信息
=====
支持获取网关的型号、MAC地址以及固件版本。

=====

.. literalinclude:: ../../../../example/cassiablue/gateway_info.py
   :language: python

API
=====

.. data:: cassiablue._gateway_type
   :type: str

   网关型号，例如 ``"M2000"``

.. data:: cassiablue._gateway_mac
   :type: str

   网关MAC地址，例如 ``"CC:1B:E0:E4:C8:94"```

.. data:: cassiablue._gateway_ver
   :type: str

   网关固件版本号，例如 ``"2.2.mp.2509031847"```
