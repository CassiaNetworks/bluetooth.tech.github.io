Retrieve gateway information

Supports retrieving the gateway’s model, MAC address, and firmware version.

Example

1import cassiablue
2
3model = cassiablue._gateway_type
4mac = cassiablue._gateway_mac
5ver = cassiablue._gateway_ver
6
7print(model, mac, ver)

API

cassiablue._gateway_type: str

Gateway model, for example "M2000"

cassiablue._gateway_mac: str

Gateway MAC address, for example "CC:1B:E0:E4:C8:94"`

cassiablue._gateway_ver: str

Gateway firmware version, for example "2.2.mp.2509031847"`
