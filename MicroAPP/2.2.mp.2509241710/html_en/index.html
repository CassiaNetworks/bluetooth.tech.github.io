Micro APP

The Cassia M2000 Bluetooth gateway now supports edge computing, enabling significantly reduced response latency, lower data transmission and cloud service costs, and enhanced overall system scalability.

Users can develop custom applications based on their specific scenarios and deploy them on the gateway. These applications share the gateway’s CPU, memory, and other computing resources, enabling local data processing and intelligent control.
